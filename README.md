# 📊 سیستم یکپارچه استخراج و تحلیل آگهی‌های شغلی (Job Market Scraper & Analyzer)

یک اپلیکیشن تحت وب (Web-based) قدرتمند که با استفاده از پایتون و Streamlit ساخته شده است. این ابزار به شما کمک می‌کند تا آگهی‌های شغلی سایت **جاب‌ویژن (JobVision)** را به صورت لحظه‌ای استخراج (Scrape) کرده و با استفاده از هوش مصنوعی و پردازش متن، بازار کار را تحلیل کنید.

## ✨ کاربردها و ویژگی‌های کلیدی

* **استخراج هوشمند و ضد-ربات:** استفاده از `undetected-chromedriver` برای دور زدن محدودیت‌های ربات‌یاب سایت‌ها و استخراج امن داده‌ها.
* **پشتیبانی از دسته‌بندی‌های متنوع:** امکان انتخاب همزمان چندین دسته‌بندی (توسعه‌دهنده، علوم داده، دیجیتال مارکتینگ، مدیریت محصول و...).
* **فیلتر زمانی دقیق:** فیلتر آگهی‌ها بر اساس زمان انتشار (امروز، ۲ روز اخیر، ۱ هفته اخیر، ۱ ماه اخیر).
* **تحلیل عمیق مهارت‌ها:** جستجو و شمارش بیش از ۴۰ مهارت تخصصی (زبان‌های برنامه‌نویسی، ابزارهای DevOps، هوش مصنوعی، دیتابیس‌ها و...) در متن آگهی‌ها.
* **سیستم تاریخچه و ذخیره خودکار (Run History):** ذخیره خودکار نتایج هر اجرا (شامل متادیتا، فایل اکسل و عکس نمودارها) در پوشه `history/` و امکان مشاهده مجدد آن‌ها از طریق پنل کناری.
* **مدیریت خودکار درایور کروم:** تشخیص خودکار مسیر `chromedriver.exe` بدون نیاز به تنظیمات دستی پیچیده.
* **خروجی‌های حرفه‌ای و پویا:** * دانلود و ذخیره خودکار داده‌های خام و جداول تحلیلی در قالب **فایل اکسل چند-صفحه‌ای (Multi-sheet Excel)**.
  * تولید نمودار تحلیل مهارت‌ها با فرمت **PNG کیفیت بالا**.

## 🧩 معماری پروژه و کدهای مستقل

این پروژه به صورت کاملاً ماژولار (Modular) طراحی شده است تا توسعه و نگهداری آن آسان باشد:

1. **فایل `utils.py` (هسته مشترک):** قلب تپنده پردازش‌ها در این فایل قرار دارد. این ماژول شامل پیکربندی لاگ‌ها، لیست جامع کلیدواژه‌ها (`KEYWORDS_POOL`)، توابع پاک‌سازی متن فارسی و استخراج اطلاعات کلیدی (مثل سابقه کار و نوع همکاری) است که توسط سایر اسکریپت‌ها فراخوانی می‌شود.
2. **برنامه اصلی (`app.py`):** داشبورد یکپارچه تحت وب با Streamlit که رابط کاربری زیبایی برای اسکرپ، تحلیل و مدیریت تاریخچه اجراها فراهم می‌کند.
3. **کد استخراج‌گر (`scraper-pro.py`):** اسکریپت مستقل برای اتصال به سایت و استخراج داده‌ها به صورت خام در فایل CSV (بدون نیاز به رابط کاربری).
4. **کد تحلیل‌گر (`job_analitico.py`):** اسکریپت مستقل برای خواندن فایل‌های CSV و تولید گزارش‌های متنی و نموداری.

## 🛠️ پیش‌نیازها (Prerequisites)

پیش از اجرای این پروژه، مطمئن شوید که موارد زیر روی سیستم شما نصب هستند:

1. **پایتون (Python):** نسخه 3.8 یا بالاتر.
2. **مرورگر Google Chrome:** آخرین نسخه.
3. **درایور کروم (ChromeDriver):** فایل `chromedriver.exe` را متناسب با نسخه کروم خود دانلود کرده و در پوشه ریشه پروژه قرار دهید (سیستم به کمک ماژول `utils.py` به صورت خودکار آن را شناسایی می‌کند).

## 🚀 راهنمای نصب و راه‌اندازی (Installation)

**۱. کلون کردن مخزن (Clone the repository):**
```bash
git clone [https://github.com/sobhandavarzani/job-vision-scraper-analyzer.git](https://github.com/sobhandavarzani/job-vision-scraper-analyzer.git)
cd job-vision-scraper-analyzer

```

**۲. ساخت و فعال‌سازی محیط مجازی (Virtual Environment):**

```bash
# ساخت محیط مجازی
python -m venv .venv
# فعال‌سازی در ویندوز
.venv\Scripts\activate
# فعال‌سازی در مک/لینوکس
source .venv/bin/activate

```

**۳. نصب کتابخانه‌های مورد نیاز:**

```bash
pip install -r requirements.txt

```

## 🖥️ روش استفاده (Usage)

برای اجرای داشبورد یکپارچه، در محیط ترمینال دستور زیر را وارد کنید:

```bash
streamlit run app.py

```

پس از اجرای این دستور، مراحل زیر را در مرورگر دنبال کنید:

1. **پنل تنظیمات:** دسته‌بندی‌های شغلی، بازه زمانی و مهارت‌های دلخواه را از سایدبار انتخاب کنید.
2. **شروع:** روی دکمه `Start Scraping & Analysis` کلیک کنید.
3. **تاریخچه (History):** از بخش Run History در سایدبار می‌توانید اجراهای قبلی را به سرعت بازخوانی و مشاهده کنید. تمامی فایل‌های اکسل و تصاویر نیز در پوشه `history/files` ذخیره می‌شوند.
