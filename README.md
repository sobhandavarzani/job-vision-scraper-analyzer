# 📊 سیستم یکپارچه استخراج و تحلیل آگهی‌های شغلی (Job Market Scraper & Analyzer)

یک اپلیکیشن تحت وب (Web-based) قدرتمند که با استفاده از پایتون و Streamlit ساخته شده است. این ابزار به شما کمک می‌کند تا آگهی‌های شغلی سایت **جاب‌ویژن (JobVision)** را به صورت لحظه‌ای استخراج (Scrape) کرده و با استفاده از هوش مصنوعی و پردازش متن، بازار کار را تحلیل کنید.

## ✨ کاربردها و ویژگی‌های کلیدی

* **استخراج هوشمند و ضد-ربات:** استفاده از `undetected-chromedriver` برای دور زدن محدودیت‌های ربات‌یاب سایت‌ها و استخراج امن داده‌ها.
* **پشتیبانی از دسته‌بندی‌های متنوع:** امکان انتخاب همزمان چندین دسته‌بندی (توسعه‌دهنده، علوم داده، دیجیتال مارکتینگ، مدیریت محصول و...).
* **فیلتر زمانی دقیق:** فیلتر آگهی‌ها بر اساس زمان انتشار (امروز، ۲ روز اخیر، ۱ هفته اخیر، ۱ ماه اخیر).
* **تحلیل عمیق مهارت‌ها:** جستجو و شمارش بیش از ۴۰ مهارت تخصصی (زبان‌های برنامه‌نویسی، ابزارهای DevOps، هوش مصنوعی، دیتابیس‌ها و...) در متن آگهی‌ها.
* **استخراج اطلاعات کلیدی:** استخراج خودکار میانگین "سال سابقه کار" درخواستی و "نوع همکاری" (دورکاری، حضوری، هیبرید).
* **داشبورد تعاملی:** نمایش نتایج در یک رابط کاربری زیبا با استفاده از نمودارهای `Seaborn` و `Matplotlib`.
* **خروجی‌های حرفه‌ای و پویا:** * دانلود داده‌های خام و جداول تحلیلی در قالب **فایل اکسل چند-صفحه‌ای (Multi-sheet Excel)**.
  * دانلود نمودار تحلیل مهارت‌ها با فرمت **PNG کیفیت بالا**.
  * نام‌گذاری خودکار و پویای فایل‌های خروجی بر اساس فیلترهای انتخابی کاربر.

## 🧩 معماری و کدهای مستقل (Important Note)

این پروژه به گونه‌ای طراحی شده است که کدهای پایه آن کاملاً ماژولار هستند. علاوه بر فایل یکپارچه `app.py` که رابط کاربری تحت وب را اجرا می‌کند، هسته اصلی سیستم از دو اسکریپت مجزا تشکیل شده است که **می‌توانند به صورت کاملاً مستقل و جداگانه نیز اجرا شوند**:

1. **کد استخراج‌گر (`scraper-pro.py`):** وظیفه اتصال به سایت، دور زدن محدودیت‌ها، جمع‌آوری لینک‌ها و استخراج متن آگهی‌ها را بر عهده دارد و در نهایت خروجی را به صورت فایل CSV ذخیره می‌کند.
2. **کد تحلیل‌گر (`job_analitico.py`):** این اسکریپت فایل‌های CSV جمع‌آوری شده را می‌خواند، کلیدواژه‌ها را استخراج می‌کند و گزارش‌های متنی و نمودارهای گرافیکی را می‌سازد.

*در واقع فایل `app.py` ترکیبی هوشمند از این دو اسکریپت است که کار با آن‌ها را از طریق رابط کاربری Streamlit برای کاربران نهایی ساده‌تر می‌کند.*

## 🛠️ پیش‌نیازها (Prerequisites)

پیش از اجرای این پروژه، مطمئن شوید که موارد زیر روی سیستم شما نصب هستند:

1. **پایتون (Python):** نسخه 3.8 یا بالاتر.
2. **مرورگر Google Chrome:** آخرین نسخه.
3. **درایور کروم (ChromeDriver):** * فایل `chromedriver.exe` را متناسب با نسخه کروم خود دانلود کرده و در پوشه پروژه قرار دهید.
   * **⚠️ نکته مهم:** در فایل‌ها، مسیر متغیر `driver_path` را به مسیر دقیق فایل ChromeDriver در سیستم خود تغییر دهید.

## 🚀 راهنمای نصب و راه‌اندازی (Installation)

**۱. کلون کردن مخزن (Clone the repository):**

```bash
git clone [https://github.com/sobhandavarzani/job-vision-scraper-analyzer.git](https://github.com/sobhandavarzani/job-vision-scraper-analyzer.git)
cd job-vision-scraper-analyzer
```
**۲. ساخت و فعال‌سازی محیط مجازی (Virtual Environment):**
```bash
# ساخت محیط مجازی
python -m venv .venv
# فعال‌سازی در ویندوز
.venv\Scripts\activate
# فعال‌سازی در مک/لینوکس
source .venv/bin/activate
```

**۳. نصب کتابخانه‌های مورد نیاز:**

```bash
pip install -r requirements.txt

```

## 🖥️ روش استفاده (Usage)

برای اجرای داشبورد یکپارچه، در محیط ترمینال (در حالی که محیط مجازی فعال است)، دستور زیر را وارد کنید:

```bash
streamlit run app.py

```

پس از اجرای این دستور، یک تب جدید در مرورگر شما باز می‌شود. مراحل کار به این شکل است:

1. **پنل تنظیمات (سایدبار):** از منوی سمت چپ، دسته‌بندی‌های شغلی مورد نظر خود را انتخاب کنید (الزامی).
2. **بازه زمانی:** مشخص کنید که آگهی‌های چه بازه زمانی اسکرپ شوند (مثلاً یک هفته اخیر).
3. **فیلتر مهارت‌ها (اختیاری):** اگر می‌خواهید فقط بازار کارِ مهارت‌های خاصی (مثل Python و React) را بسنجید، آن‌ها را انتخاب کنید.
4. **شروع:** روی دکمه `Start Scraping & Analysis` کلیک کنید.
5. **دریافت خروجی:** پس از پایان استخراج و نمایش نمودارها، می‌توانید فایل اکسل و عکس نمودار را از طریق دکمه‌های پایین صفحه دانلود کنید.
